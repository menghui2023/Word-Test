👨‍🎓 如何進行測驗？

選擇測驗範圍：
進入網頁後，點擊上方藍色區塊的下拉選單，選擇你要測驗的單字單元（例如：三民 Level 3）。選擇後，系統會在一秒內自動產生一份全新試卷。

開始作答：

聽英拼寫： 點擊喇叭圖示聽發音，用「全小寫」拼出單字。

詞彙題： 包含單句選擇、單句配對及短文克漏字，請依據句意選出正確答案。

交卷與檢討：
作答完畢後，滑到網頁最下方點擊紅色的 [📝 交卷並查看成績]。接著點擊 [👨‍🏫 顯示測驗詳解]，即可查看完整的中文翻譯、例句與錯誤解析。

查看紀錄：
